There are a couple of ways to go about this, but I'll give you the simplest.

The main thing is that `me` is not the first parameter in your function call;
the first argument in your function call is the index passed by Python when
it calls the `get_index()` method. To pass an argument to the function, we need to
create a new object that has `self` as its first attribute and then assign
itself (i.e., the parameter we want to pass) to its `self._name` property:

```python
def get_index(self):
    return self._name[0]  # pass 'me' as argument to function
```

Now when you call this method on our `Person` class, we can simply use the index
of that attribute in order to retrieve it. Let me show you an example:

```python
my_person = Person('John Smith', age=12)  # create a new person object
index = my_person.get_index()            # get the name of this person
print(f"The index of {my_person} is: {index}")   # output: The index of John Smith is: me
```

The above example simply prints out that `me` is the index of `John Smith`, as expected.
