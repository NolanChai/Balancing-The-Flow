### User Story:
As a user, I want to be able to check whether someone is cheating on me or not.

### Features:
* The app should have a simple interface where a user can type in the name of their partner and click on "Check" button.
* The app should return back a message saying if they are cheating or not.
* If they are, then it should give a link to a picture of them with another person.

### Technologies used:
* HTML5
* CSS
* Javascript
* Heroku
* Bootstrap

### Testing:
1. Click on "Check" button, enter a name and click on "Submit". The message should appear saying that they are cheating or not. If they are, then it will give a link to the photo of them with another person.
2. Enter a different name and click on "Check" again. The result should change.
3. If you click on "Cheating?" button, the message should appear saying that they are cheating or not.
4. If you click on "Photo" button, it will show the photo of them with another person.
