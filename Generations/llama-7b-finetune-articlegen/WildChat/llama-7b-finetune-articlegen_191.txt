The first thing to do is to convert the integer value into a string. The problem with this code is that it reads the arguments as strings, not integers, so we have to cast them back into ints:
```python
white_balls = [int(sys.argv[i]) for i in range(2, 7)]
# or better yet...
white_balls = list(map(int, sys.argv[2:])) # you can also use .split() and strip() to remove the whitespace
```
Once you have your values, it is simple math from there (though I would probably add a little more error checking than just len(sys.argv) != 7):
```python
def calculate_result(white_balls, power_ball):
	# white_balls: list of integer values
	# power_ball: int
	return white_balls[0] + (white_balls[1] * white_balls[2]) + ... + (white_balls[-1] * white_balls[-2]) + ((power_ball - 1) * (len(white_balls)-2))
```

### Explanation:
The difference is in the mode of the files:

	-rwxr-xr-x : Owner may read, write and execute
	-rw-r--r-- : Everyone else can read
	drwxr-xr-x  : Owner may create directories but not files
	drwxr-xr-x   : The owner may delete the file or directory if it is the only one in a directory.  Otherwise, no one except root may remove this file or folder (not even the owner).

### Note:
The `chmod` command does not change permissions of existing files/folders, but instead changes how they are displayed to other people.
