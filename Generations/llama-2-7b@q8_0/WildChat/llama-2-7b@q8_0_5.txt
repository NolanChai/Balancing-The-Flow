## How to run your prompt

To run your prompt, follow these steps.

1. Run `python3 main.py` to open a command-line interface (CLI).
2. Type `prompt_generator -h` to see the available command-line options and usage information.
3. Follow the prompts to enter your own concept prompts.
4. Once you have finished, type `quit` to exit.
