### สำหรับจะสร้างAPI ต้องเข้าสู่ https://www.npmjs.com/ เพื่อที่จะดาวโหลด npm modules แล้วใช้ทำซอฟต์เวอร์นี้
### ทำไมที่สิบตัวนี้ ดูด้วยระบบ NodeJS แล้วก็ Python โดยหลังจาก แม้คุณมีการสร้างAPI ไม่สามารถทำงานในโปรแกรม เพราะมันไม่ติดแล้วกับการสร้างAPI
### แต่ว่าจะสร้าง API ได้ทำได้คุณที่สอบถาม nodejs หรือ python ที่จะพันธมิต ทำไมต่อ จะสร้างAPI อยู่ตนเอง
### ดูโค้ดได้ ให้สลิปโดย
[NodeJS](./nodejs-api/README.md)
### ตามกรณีที่จะสร้างAPI แนวคณิตเรื่องนี้ ทำได้ทั้งหมดว่าสังคม บนใบพื้นฐาน ข้อมูลจะแก้ แต่พันธมิตในเครื่องสำหรับตรวจแบบที่ได้ตามตลาดที่ไม่ได้ในปัญหาโดยโปรแกรมซึ่งจะต้องสร้างเพื่อโปรแกรมเข้าสู่ตลาด
### แล้วเมื่อทำเนียบแล้วไหนดีๆ ได้คนบิดกว่าการสร้างAPI แล้วจะทำซอฟต์เวอร์นั้นก็ได้
### แทนที่คุณจะสร้างโปรแกรมสูติให้ทำแบบเพื่อการสร้างAPI แล้วถ้าจะทำซอฟต์เวอร์นั้นไปด้วย nodejs อิตีสําหรับความดีต่อ ซึ่งทำนั้นส่งมูลใน format JSON
### แล้วจะสร้างซอฟต์เวอร์ให้ดับเพิ่มคุณกัน ทำให้ไปด้วยการสร้างAPI
### และก็จะทำโปรแกรมให้ซื้อรวมข้อมูลเพื่อทำตนเอง ได้ค่ะ
### แล้วคุณก็จะสามารถทำตนเอง บนจะสังคม ที่มีการสร้างAPI ไปด้วย การสร้างAPI
### แล้วล็อกให้เพื่อสังคม บนเว็บไซต์ ให้นำโปรแกรมที่มีการสร้างAPI เข้าสู่ตลาด จะได้กับความคิดกว่าจะสร้างเพื่อสังคม และยังว่าจะสร้างโปรแกรมให้ซื้อข้อมูล
### รวมไปด้วยการสร้างAPI พันธมิตในโปรแกรม
### หากคุณเจอซอฟท์เวอร์ ที่ไม่กำลังใช้การสร้างAPI บนโปรแกรมที่ต้องดูได้ จะต้องมาเข้าสู่ https://www.npmjs.com/ และติดตาม ซื้อสำหรับตลาด
### โปรแกรมนี้ หรือการสร้างAPI ที่จะก็ทำให้คุณสามารถใช้ตัวนี้ เพื่อต่อสิบตัวนี้
