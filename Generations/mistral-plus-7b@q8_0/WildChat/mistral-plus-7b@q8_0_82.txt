1. 第一个问题：不能用%s，应该用?替换

2. sql语句中的字段名必须用单引号括起来，`%s`是字符串格式化，是lua语言提供的一种写法，但是sql里不需要这样写。如果这么写了，那么在mysql中会报错：You have an error in your SQL syntax; check the manual that corresponds to your MySQL server version for the right syntax to use near '%s' at line 1.

3. 如果用?替换字段名，并且字段类型是数字类型，那么语句应该写成update club_info set %s=? where cid=?。这里的cid必须在最后，因为参数匹配时按照顺序进行的，所以在后面放cid才能够和cid匹配上。

4. 如果字段类型是字符串，那么就要用string.format()格式化，使用%s替换字段名。例如：local sql = string.format("update club_info set %s=\"%s\" where cid=%d", field_name, value, cid)

总结：sql语句中的字段名应该用单引号括起来，并且使用?或者%s替换。如果使用?替换，那么字段类型是数字时要放在最后；如果使用%s替换，那么字段类型是字符串时就要用string.format()格式化。
