This project will allow you to automate ping command using c# . The code can be modified with the help of the source file. You could modify it as per your needs. This tool is used for network automation testing. It allows us to check the connectivity of a host by sending ICMP Echo Request packets and waiting for an ICMP Echo Reply packet.

When we run this code , we will have to enter the IP Addresses whose connectivity has to be tested, then it will start pinging that IP Addresses . It will display the result and will also log the results into a text file.
